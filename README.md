# ⛩️ Code-Dojo

> _"Your digital dojo to master the art of code. Discover, filter, and collect sacred scrolls of knowledge without the overwhelm."_

🔗 **Live Demo:** [code-dojo.vercel.app](https://code-dojo-inky.vercel.app/) ·
📁 **Repo:** [github.com/rianaripn/CodeDOJO](https://github.com/rianaripn/CodeDOJO)

## Features

- **The Training Grounds** — Curated resources grouped by tech stack (React, JavaScript, CSS, Node.js)
- **Rank Filtering** — Filter techniques by rank: Kohai (Beginner), Senpai (Intermediate), Sensei (Advanced)
- **Jutsu Details** — Dedicated detail page per resource with description, source link, and estimated training time
- **Sacred Scrolls** — Bookmark system via `localStorage` — no login required, yours alone to revisit

---

## Tech Stack

| Layer       | Tech                         |
| ----------- | ---------------------------- |
| Framework   | React.js + Vite              |
| Routing     | React Router v6              |
| Styling     | Tailwind CSS v4 + Custom CSS |
| Data        | Static JSON                  |
| Persistence | localStorage                 |
| Deployment  | Vercel                       |

---

## Getting Started

```bash
# Clone the repo
git clone https://github.com/rianaripn/CodeDOJO.git

# Install dependencies
cd CodeDOJO
npm install

# Run locally
npm run dev
```
